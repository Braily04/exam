Question and Answer

1. How to create reactive forms?

ANSWER: To create reactive forms in Angular, import ReactiveFormsModule and use FormGroup and FormControl to define the form's structure and validation, offering greater control over form behavior.

2. How to add built-in validation for email reactive form?

ANSWER: To add built-in email validation in a reactive form, use the Validatiors.email along with Validators.required when defining the form control.

3. How to add built-in validation for required fields in reactive form?

ANSWER: To add built-in validation for required fields in a reactive form, use the Validators.required when defining the form control.

4. How to add regex validation for reactive forms?

ANSWER: To add regex validation in a reactive form, use Validators.pattern with the desired regular expression when defining the form control.

5. How to generate routing?

ANSWER: To generate routing in Angular, use the command ng generate module app-routing --flat --module=app to create a routing module and configure it in your application.

6. How to configure routes with parameters?

ANSWER: To configure routes with parameters, define a route with a parameter in the path, like path: 'example/:id', and retrieve it in the component using Angular's AtivatedRoute.

7. What is DebounceTime in RxJS?

ANSWER: debounceTimeinRxJS is an operator that delays the emission of values from an observable by a specified time, emitting only the most recent value within that timeframe.

8. What is pipe operator?

ANSWER: The pipe operatoris a method used to combine multiple operators in a readable and functional way, enabling transformation or handling of observable data.

9. What is distinctUntilChanged operator?

ANSWER: The distinctUntilChanged operator in RxJS filters out consecutive, emitting only when the current value differs from the previous one.

10. What is filter operator in RxJS?

ANSWER: The filter operator in RxJS allows you to emit values from an observable only if they satisfy a specified condition ot predicate.
